# OI-codes
我的一些 OI（信息学竞赛）学习资料

这里包含
- 一些我做过的信息学竞赛题目的代码
- 一些算法与数据结构的模板代码
- 一些算法与数据结构的介绍

大部分代码和一些介绍是我自己写的，另一些介绍是我觉得比较好的资料，均已注明出处（假如我找得到），一部分模板代码参考了一些教材与资料。

所有的资料均已按照算法 / 赛事分类（其中2019年之后赛事的代码按照赛事分类）。
